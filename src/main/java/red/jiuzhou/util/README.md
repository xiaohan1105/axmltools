# 工具类

本包包含应用程序中使用的各种工具类。

## 数据库工具
- `DatabaseUtil.java` - 数据库连接和实用功能
- `SqlGeneratorUtil.java` - SQL 生成工具

## 配置工具
- `YamlUtils.java` - YAML 配置工具
- `YmlConfigUtil.java` - YAML 配置管理

## 文件处理
- `XmlUtil.java` - XML 处理工具
- `PathUtil.java` - 文件路径工具
- `MultiXMLMerger.java` - 多 XML 文件合并
- `XmlStringModifier.java` - XML 字符串修改

## 数据处理
- `JSONRecord.java` - JSON 记录处理
- `Recordset.java` - 数据集处理
- `CounterUtil.java` - 进度计数工具

## 外部服务
- `AliyunTranslateUtil.java` - 阿里云翻译服务集成
- `IncrementalMenuJsonGenerator.java` - 动态菜单生成

## 关键功能

- 数据库连接管理
- 多格式文件处理（XML、JSON、YAML）
- 翻译服务集成
- 进度跟踪和报告
- 文件系统工具
