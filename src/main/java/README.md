# Java 源代码

此目录包含 `red.jiuzhou` 包下的所有 Java 源代码。

## 包结构

- `red.jiuzhou.ui` - 用户界面组件
- `red.jiuzhou.dbxml` - 数据库-XML转换核心
- `red.jiuzhou.xmltosql` - XML到SQL生成
- `red.jiuzhou.util` - 工具类
- `red.jiuzhou.api` - API层
- `red.jiuzhou.tabmapping` - 表映射
- `red.jiuzhou.ai` - AI集成

## 主要组件

### UI包
基于JavaFX的桌面应用程序界面，具有选项卡导航、数据表和各种对话框窗口。

### 核心包
数据库-XML转换引擎、XML处理工具和SQL生成工具。

### AI集成
多个AI服务集成，包括Qwen、Doubao、Kimi和DeepSeek，用于内容处理和翻译。
